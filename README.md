# 🧠 Socratic Coding Mentor

> **Active Learning, not just Auto-Complete.**

Socratic Coding Mentor is a high-UI rich Chrome Extension designed to bridge the gap between solving a coding problem and truly understanding the logic. Built for platforms like **LeetCode** and **CodeChef**, it provides a "Virtual Debugger" experience using ultra-fast AI inference.

---

## ✨ Key Features

* **⚡ Groq-Powered Simulation:** Uses Llama 3.3 via Groq LPUs for sub-second execution tracing.
* **🧩 Memory Map Visualization:** Renders variable states as sleek, minimalistic cards that update as you step through code.
* **🔍 Socratic Guidance:** Instead of giving solutions, the mentor identifies logical fallacies and asks guiding questions.
* **🛡️ Shadow DOM Encapsulation:** Zero CSS interference with host coding platforms.
* **🎛️ Minimalist Draggable UI:** A sophisticated, GitHub-dark inspired interface that stays in your flow.

## 🛠️ Tech Stack

* **Frontend:** React 18, Vite, Tailwind CSS
* **Extension Framework:** CRXJS
* **AI Orchestration:** OpenAI SDK (Groq Cloud)
* **State Management:** React Hooks + MutationObserver API
